# Hero Images

Drop hero portrait images here to enable real hero art in the UI.

## Naming
Use lowercase kebab-case hero names:

- `abrams.png`
- `grey-talon.png`
- `lady-geist.png`
- `mo-and-krill.png`
- `the-doorman.png`

## Supported formats
- `.png`
- `.jpg`
- `.webp`

If an image is missing, the app uses the bundled placeholder avatar.
